Are they macroscopic?

Macroscopicity

Behavior is neither
classical, nor an
accumulated quantum effectThink Debye model for specific heat.

Idea: Use effective number of particles The smallest system size so that quantum mechanics explains the behavior. Think Neff = 2 for Cooper pairs.
Call a system macroscopic if effective size is linear Neff = $\mathcal O$ (N) I.e. all particles need to be described quantum mechanically.

Examples:

GHZ state: Neff = N, macroscopic

Cooper pair: Neff = 2, microscopic
